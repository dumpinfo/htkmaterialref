5.8.2 Esignal File Format

  The Esignal file format for waveforms is similar to that described in section 5.7 above with the following exceptions. When reading an ESIG waveform file the HTK programs only check whether the record length equals 2 and whether the datatype of the only field in the data records is SHORT. The data field that is created on output of a waveform is called WAVEFORM.
