13.10.4 Tracing

HLSTATS supports the following trace options where each trace flag is given using an octal base

    00001
    basic progress reporting.
    00002
    trace memory usage.
    00004
    show basic statistics whilst calculating bigrams. This includes the global training data entropy and the entropy for each distinct label.
    00010
    show file names as they are processed.
Trace flags are set using the -T option or the TRACE configuration variable.
