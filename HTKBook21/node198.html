13.6.3 Tracing

HHED supports the following trace options where each trace flag is given using an octal base

    00001
    basic progress reporting.
    00002
    intermediate progress reporting.
    00004
    detailed progress reporting.
    00010
    show item lists used for each command.
    00020
    show memory usage.
    00100
    show changes to macro definitions.
    00200
    show changes to stream widths.
    00400
    show clusters.
    00800
    show questions.
    01000
    show tree filtering.
    02000
    show tree splitting.
    04000
    show tree merging.
    10000
    show good question scores.
    20000
    show all question scores.
    40000
    show all merge scores.
Trace flags are set using the -T option or the TRACE configuration variable.
