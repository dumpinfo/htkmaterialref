13.7.3 Tracing

HINIT supports the following trace options where each trace flag is given using an octal base

    000001
    basic progress reporting.
    000002
    file loading information.
    000004
    segments within each file.
    000010
    uniform segmentation.
    000020
    Viterbi alignment.
    000040
    state alignment.
    000100
    mixture component alignment.
    000200
    count updating.
    000400
    output probabilities.
Trace flags are set using the -T option or the TRACE configuration variable.
