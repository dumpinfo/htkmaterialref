13.11.5 Tracing

HPARSE supports the following trace options where each trace flag is given using an octal base

    0001
    basic progress reporting.
    0002
    show final HParse network (before conversion to a lattice)
    0004
    print memory statistics after HParse lattice generation
    0010
    show progress through glue node removal.
Trace flags are set using the -T option or the TRACE configuration variable.
